# Student Performance Analysis

This project focuses on analyzing student academic performance using Python and data visualization techniques.

## Project Objective

The goal of this project is to identify the key factors that influence final student grades (G3) and understand how study habits, failures, and attendance affect academic performance.

## Tools Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook

## Key Analysis Performed

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Final grade distribution analysis
- Study time vs final grade analysis
- Failures vs final grade analysis
- Correlation heatmap visualization

## Key Insights

- Previous grades (G1, G2) strongly influence final grades (G3)
- Students with more past failures tend to score lower
- Study time shows a positive relationship with academic performance
- Absences have a moderate impact on final outcomes
